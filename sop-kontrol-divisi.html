2. TUGAS POKOK PER DIVISI (各部門主要職責)

Marketing / 行銷部

  • Akuisisi anggota baru melalui grup LINE/WhatsApp
  • Kumpulkan dokumen aplikasi pinjaman
  • Buat Memo Awal untuk Checking
  • Jaga hubungan dengan agen/majikan di Taiwan
  • Field visit ke TKW secara berkala

Checking & Administrasi / 審核部

  • Verifikasi dokumen asli vs fotokopi
  • Cek histori pinjaman & BI Checking
  • Analisa 5C: Character, Capacity, Capital, Collateral, Condition
  • Hitung plafon maksimal (angsuran max 40% gaji)
  • Buat Surat Keputusan Kredit (SK-Kredit)

Keuangan & Akuntansi / 財務會計部

  • Pembukuan harian: jurnal, buku besar, GL
  • Pencairan dana pinjaman (transfer)
  • Rekonsiliasi bank & kas mingguan
  • Laporan keuangan bulanan (Neraca, LR, AR)
  • Kepatuhan pajak: PPh 21/23/4(2), PPN

Penagihan / 催收部

  • Catat jadwal angsuran & jatuh tempo
  • Kirim reminder H-3 via LINE/WhatsApp
  • Penagihan bertahap: SP1, SP2, SP3
  • Restrukturisasi pinjaman jika perlu
  • Recovery pinjaman macet

3. ALUR KONTROL DIVISI (部門控制流程)

3.1. Alur Pinjaman (貸款流程)

TAHAP 1 — MARKETING 行銷部
Promosi ke TKW via LINE/WhatsApp → Isi formulir → Kumpulkan dokumen (KTP, Passport, Visa, Kontrak Kerja, Slip Gaji) → Buat Memo Awal
Target: ≤ 1 hari kerja
↓
TAHAP 2 — CHECKING 審核部
Input data ke sistem → Verifikasi dokumen → Cek histori & BI Checking → Analisa 5C → Hitung plafon maks (angsuran ≤ 40% gaji, tenor ≤ sisa kontrak)
Target: ≤ 3 hari kerja
↓
TAHAP 3 — KEPUTUSAN KREDIT 核貸決定
≤ Rp5jt: Manajer | Rp5–15jt: Manajer + Pengurus | ≥ Rp15jt: Rapat Pengurus
Jika ditolak: Marketing informasikan. Jika disetujui: Kirim SK-Kredit ke Keuangan
Target: ≤ 1 hari kerja
↓
TAHAP 4 — KEUANGAN 財務部
Cek SK-Kredit & dokumen → Jurnal pencairan → Transfer ke rekening Taiwan/Indonesia → Simpan bukti transfer + tanda tangan
Target: ≤ 1 hari kerja setelah SK-Kredit
↓
TAHAP 5 — PENAGIHAN 催收部
Catat jatuh tempo → H-3 reminder LINE/WA → H+1 telpon → H+7 SP1 → H+14 SP2 + hubungi keluarga → H+30 SP3 / restrukturisasi
Target Collection Rate: ≥ 95% bulanan

5. PENCAPAIAN TARGET (達成目標)

Target Utama Koperasi:
1. NPL ≤ 5% — melalui kontrol ketat checking + penagihan disiplin
2. Collection rate ≥ 95% — reminder terjadwal + follow up bertahap
3. Pertumbuhan portofolio ≥ 20% per tahun — via marketing aktif
4. Laporan keuangan akurat & tepat waktu — rekonsiliasi harian + audit trail
5. Kepatuhan pajak 100% — jadwal setor dan lapor terintegrasi

Roda kontrol (control wheel):
Marketing (akuisi) → Checking (verifikasi) → Keuangan (pencairan + pembukuan) → Penagihan (pemantauan) → Marketing (hubungan anggota) — siklus tertutup dengan audit trail di setiap tahap.
