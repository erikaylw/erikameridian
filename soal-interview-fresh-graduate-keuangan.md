## BAGIAN A — 15 Soal Wawancara + Jawaban

### 1. Ceritakan tentang diri Anda.
> Saya lulusan [jurusan] dari [universitas] dengan IPK [nilai]. Selama kuliah saya aktif di [organisasi/magang] yang mengasah kemampuan administrasi dan ketelitian saya. Saya tertarik di bidang keuangan karena saya suka bekerja dengan angka dan memastikan semuanya akurat. Saya orangnya terstruktur, disiplin, dan cepat belajar software baru.

### 2. Apa yang Anda ketahui tentang siklus akuntansi?
> Siklus akuntansi dimulai dari pencatatan transaksi di jurnal umum, posting ke buku besar, penyusunan neraca saldo, jurnal penyesuaian, neraca saldo disesuaikan, laporan keuangan (Laba Rugi, Perubahan Ekuitas, Neraca, Arus Kas), dan diakhiri dengan jurnal penutup.

### 3. Bedakan antara utang usaha (AP) dan piutang usaha (AR).
> Utang usaha (Accounts Payable) adalah kewajiban perusahaan kepada pemasok atas pembelian barang/jasa secara kredit. Piutang usaha (Accounts Receivable) adalah hak perusahaan untuk menagih pelanggan atas penjualan secara kredit. AP ada di sisi pasiva neraca, AR di sisi aset lancar.

### 4. Bagaimana cara Anda memastikan data keuangan akurat?
> Saya biasakan double-check setiap entry, melakukan rekonsiliasi bank secara rutin, dan membandingkan data dengan dokumen sumber (invoice, kwitansi, kontrak). Saya juga menggunakan pivot table atau VLOOKUP di Excel untuk mencocokkan data antar sheet sebelum melapor.

### 5. Apa yang dimaksud dengan PPh 21? Jelaskan secara sederhana.
> PPh 21 adalah pajak penghasilan yang dipotong oleh perusahaan dari gaji karyawan setiap bulan dan disetorkan ke negara. Tarifnya progresif — semakin besar penghasilan, semakin besar persentase pajaknya. Perusahaan wajib memotong, menyetor, dan melaporkan paling lambat tanggal 10 bulan berikutnya.

### 6. PPN itu apa dan siapa yang memungutnya?
> PPN (Pajak Pertambahan Nilai) adalah pajak atas konsumsi barang/jasa di dalam daerah pabean. Pengusaha Kena Pajak (PKP) memungut PPN 11% (tarif 2026) dari pelanggan, kemudian menyetorkan selisih antara PPN keluaran dan PPN masukan ke kas negara.

### 7. Jelaskan perbedaan antara laba kotor dan laba bersih.
> Laba kotor = Penjualan bersih dikurangi Harga Pokok Penjualan (HPP). Laba bersih = Laba kotor dikurangi semua beban operasional, pajak, dan bunga. Laba bersih adalah angka bottom-line yang menunjukkan keuntungan sebenarnya.

### 8. Anda menemukan selisih Rp 50.000 di buku kas. Apa yang Anda lakukan?
> Pertama, saya cek ulang transaksi hari itu satu per satu — mungkin ada kesalahan input atau transaksi terlewat. Kalau tidak ketemu, saya cek saldo fisik dan rekonsiliasi bank. Kalau selisih tetap tidak terjawab, saya konsultasi ke atasan dan catat di jurnal selisih kas (cash difference) dengan otorisasi yang tepat.

### 9. Apa itu rekonsiliasi bank?
> Rekonsiliasi bank adalah proses mencocokkan saldo kas di catatan perusahaan dengan saldo yang tercatat di rekening bank. Perbedaan biasanya karena setoran dalam perjalanan (deposit in transit), cek beredar (outstanding checks), biaya bank, atau bunga yang belum dicatat.

### 10. Software akuntansi apa yang pernah Anda gunakan?
> [Jawab sesuai pengalaman — misal: Saya pernah menggunakan Accurate, Zahir, dan Excel intensif selama magang. Saya juga belajar dasar-dasar SAP dan Odoo di kampus. Saya yakin bisa cepat adaptasi dengan software baru karena saya terbiasa belajar dari tutorial dan dokumentasi.]

### 11. Menurut Anda, soft skill apa yang paling penting untuk admin keuangan?
> Ketelitian (attention to detail) nomor satu — satu digit salah bisa menyebabkan laporan salah. Kedua, kemampuan mengatur waktu karena banyak deadline (setoran pajak, closing bulanan). Ketiga, komunikasi — admin keuangan harus bisa menjelaskan masalah angka ke bagian lain dengan bahasa yang jelas.

### 12. Bagaimana cara Anda menangani tekanan deadline closing bulanan?
> Saya selalu membuat checklist closing harian minggu terakhir bulan itu. Prioritas: entry invoice dan pengeluaran lebih awal, rekonsiliasi di pertengahan, review jurnal penyesuaian di akhir. Saya juga komunikasi ke tim lain (sales, purchasing) untuk memastikan dokumen masuk tepat waktu.

### 13. Apa itu jurnal penyesuaian? Beri contoh.
> Jurnal penyesuaian adalah jurnal yang dibuat di akhir periode untuk mencatat transaksi yang belum tercatat atau alokasi yang belum tepat. Contoh: beban gaji yang masih harus dibayar — kita catat beban gaji (debit) dan utang gaji (kredit) meskipun gaji belum dibayar.

### 14. Bagaimana pendapat Anda tentang kerahasiaan data keuangan?
> Data keuangan adalah informasi sensitif perusahaan. Saya paham konsekuensi hukum dan etika dari pembocoran data. Saya akan menjaga kerahasiaan penuh, termasuk tidak membahas angka gaji atau laporan keuangan dengan pihak yang tidak berkepentingan, bahkan dengan rekan di luar departemen.

### 15. Apa rencana karir Anda 3 tahun ke depan?
> Tahun pertama saya targetkan menguasai proses operasional harian — AP, AR, bank reconciliation, pajak dasar. Tahun kedua saya ingin ambil sertifikasi Brevet A&B pajak. Tahun ketiga saya mulai belajar analisis laporan keuangan dan budgeting, sehingga bisa naik level ke posisi staff finance yang lebih strategis.

### Sheet 2: Daftar Gaji & PPh 21

Buat tabel:

| No | Nama | Gaji Pokok | Tunjangan | Bruto | PTKP (TK/0) | PKP | PPh 21 Sebulan |
|----|------|-----------|-----------|-------|-------------|-----|----------------|
| 1  | Andi | 5,000,000 | 1,000,000 |       | 54,000,000  |     |                |
| 2  | Budi | 4,200,000 | 500,000   |       | 54,000,000  |     |                |
| 3  | Cici | 6,000,000 | 1,500,000 |       | 54,000,000  |     |                |
| 4  | Dedi | 3,500,000 | 300,000   |       | 54,000,000  |     |                |
| 5  | Eka  | 7,500,000 | 2,000,000 |       | 54,000,000  |     |                |

**Rumus:**
- Bruto = Gaji Pokok + Tunjangan
- PTKP per tahun = Rp 54.000.000 (TK/0 untuk lajang tanpa tanggungan)
- Penghasilan Neto setahun = (Bruto x 12) - (Biaya Jabatan 5% x Bruto x 12, maks 500.000/bulan atau 6.000.000/tahun)
- PKP setahun = Penghasilan Neto setahun - PTKP
- PPh 21 setahun:
  - 5% x PKP (s.d. Rp 60.000.000)
  - 15% x kelebihan PKP di atas Rp 60.000.000 (s.d. Rp 250.000.000)
- PPh 21 sebulan = PPh 21 setahun / 12

**Soal:**
1. Isi semua kolom yang kosong menggunakan rumus Excel.
2. Format rupiah pada semua kolom angka.
3. Berapa total PPh 21 yang harus disetor perusahaan untuk 5 karyawan ini?

## BAGIAN C — Kunci Jawaban Test Excel

### Sheet 1 — Jawaban

1. **Total Debit** = 500rb + 750rb + 4,2jt + 200rb + 450rb + 1,1jt = **Rp 7.200.000**
   **Total Kredit** = 2,5jt + 1,8jt + 3jt + 950rb = **Rp 8.250.000**
   Tidak sama — ini adalah data arus kas, bukan jurnal double-entry.

2. **Saldo Akhir** = Rp 7.200.000 - Rp 8.250.000 = **-Rp 1.050.000** (defisit / pengeluaran > pemasukan)

3. **SUMIF per kategori:**
   - BBM (BBM-001 + BBM-002 + BBM-003): 500rb + 200rb + 1,1jt = **Rp 1.800.000**
   - INV (INV-001 + INV-002 + INV-003): 2,5jt + 1,8jt + 950rb = **Rp 5.250.000**
   - BYR (BYR-001 + BYR-002): 750rb + 450rb = **Rp 1.200.000**
   - GJI (GJI-001): **Rp 4.200.000**
   - TRF (TRF-001): **Rp 3.000.000**

### Sheet 2 — Jawaban (Perhitungan)

**Andi:**
- Bruto = 5jt + 1jt = 6.000.000
- Bruto setahun = 6jt x 12 = 72.000.000
- Biaya jabatan = min(5% x 72jt, 6jt) = 3.600.000
- Penghasilan neto setahun = 72jt - 3,6jt = 68.400.000
- PKP = 68,4jt - 54jt = 14.400.000
- PPh 21 setahun = 5% x 14,4jt = 720.000
- **PPh 21 sebulan = 720rb / 12 = 60.000**

**Budi:**
- Bruto = 4,2jt + 500rb = 4.700.000
- Bruto setahun = 56.400.000
- Biaya jabatan = min(5% x 56,4jt, 6jt) = 2.820.000
- Neto setahun = 56,4jt - 2,82jt = 53.580.000
- PKP = 53,58jt - 54jt = 0 (negatif, jadi 0)
- **PPh 21 sebulan = 0** (tidak kena pajak)

**Cici:**
- Bruto = 6jt + 1,5jt = 7.500.000
- Bruto setahun = 90.000.000
- Biaya jabatan = min(5% x 90jt, 6jt) = 4.500.000
- Neto setahun = 90jt - 4,5jt = 85.500.000
- PKP = 85,5jt - 54jt = 31.500.000
- PPh 21 setahun = 5% x 31,5jt = 1.575.000
- **PPh 21 sebulan = 1.575.000 / 12 = 131.250**

**Dedi:**
- Bruto = 3,5jt + 300rb = 3.800.000
- Bruto setahun = 45.600.000
- Biaya jabatan = min(5% x 45,6jt, 6jt) = 2.280.000
- Neto setahun = 45,6jt - 2,28jt = 43.320.000
- PKP = 43,32jt - 54jt = 0
- **PPh 21 sebulan = 0**

**Eka:**
- Bruto = 7,5jt + 2jt = 9.500.000
- Bruto setahun = 114.000.000
- Biaya jabatan = min(5% x 114jt, 6jt) = 5.700.000
- Neto setahun = 114jt - 5,7jt = 108.300.000
- PKP = 108,3jt - 54jt = 54.300.000
- PPh 21 setahun = 5% x 54,3jt = 2.715.000
- **PPh 21 sebulan = 2.715.000 / 12 = 226.250**

**Total PPh 21 disetor perusahaan per bulan** = 60rb + 0 + 131.250 + 0 + 226.250 = **Rp 417.500**

### Sheet 3 — Jawaban

1. **Total per baris** = Qty x Harga Satuan (sudah langsung dihitung)

2. **Pivot Table — Total per Pelanggan:**
   - PT ABC: (50x5rb) 250rb + (30x8rb) 240rb + (60x2,5rb) 150rb + (20x7,5rb) 150rb = **Rp 790.000**
   - CV XYZ: (100x2,5rb) 250rb + (40x7,5rb) 300rb + (80x5rb) 400rb = **Rp 950.000** ← TERBESAR
   - Toko Sejahtera: (75x5rb) 375rb + (25x8rb) 200rb + (45x2,5rb) 112.500 = **Rp 687.500**

3. **VLOOKUP** — tabel referensi harga:
   ```
   Buku Tulis  | 5,000
   Pulpen      | 2,500
   Map         | 8,000
   Stabilo     | 7,500
   ```

4. **IF Diskon:**
   Baris dengan Total > 400rb: PT ABC item Buku Tulis (250rb — tidak), Map (240rb — tidak)... CV XYZ Pulpen (250rb — tidak), Stabilo (300rb — tidak), Buku Tulis (400rb — tidak). Toko Sejahtera Buku Tulis (375rb — tidak).
   
   **Tidak ada baris yang > 400.000** — jadi semua diskon = 0%. (Ini sengaja untuk lihat apakah kandidat sadar logika IF-nya bekerja dengan benar meskipun hasilnya nol.)
