# Soal Test — Penggajian untuk Interview HRD / Finance

## Petunjuk Penguji

Dokumen ini bisa digunakan untuk:
- Interview tatap muka (bagian A & B, tanya jawab)
- Tertulis / take-home test (bagian B, jawab singkat)
- Excel test praktek (bagian C, waktu 30-45 menit)

Durasi total jika semua bagian dipakai: 60-75 menit
Passing grade: 65/100

---

## Bagian A — Pertanyaan Behavioral & General (Bobot 20%)

### A1. Perkenalan dan Pemahaman Gaji
Tanya: "Ceritakan pengalaman kamu terkait proses penggajian, baik saat magang, kerja sebelumnya, atau pelajaran di kuliah."
Amati:
- Memahami siklus penggajian (lihat → hitung → bayar → lapor) [skor 2]
- Menyebutkan komponen gaji pokok, tunjangan, potongan [skor 2]
- Jika belum punya pengalaman, antusiasme belajar [skor 1]

### A2. Ketelitian Data Gaji
Tanya: "Data gaji 50 karyawan harus selesai sore ini. Kamu menemukan selisih Rp 250.000 di total gaji. Apa yang kamu lakukan?"
Amati:
- Tidak panik, cek ulang satu per satu [skor 2]
- Pakai cara sistematis (filter, sortir, cek rumus) [skor 2]
- Berani minta bantuan atasan jika mentok [skor 1]

### A3. Kerahasiaan Gaji
Tanya: "Teman sekantor bertanya berapa gajimu. Juga minta lihat slip gaji karyawan lain. Bagaimana sikapmu?"
Amati:
- Paham gaji adalah informasi rahasia [skor 2]
- Tidak memberikan data gaji siapapun [skor 2]
- Tahu sanksi kebocoran data karyawan [skor 1]

### A4. Deadline Penggajian
Tanya: "Tanggal 25 adalah deadline gaji, tapi data absensi dari divisi A belum masuk sampai jam 3 sore. Apa yang kamu lakukan?"
Amati:
- Proaktif hubungi divisi A [skor 1]
- Punya SOP/backup plan [skor 2]
- Alternatif: gaji sesusun dulu sisanya nyusul [skor 1]
- Komunikasi ke atasan / HRD [skor 1]

### A5. Multitasking Akhir Bulan
Tanya: "Akhir bulan kamu harus: (1) siapkan gaji, (2) rekonsiliasi bank, (3) siapkan faktur pajak. Bagaimana prioritasmu?"
Amati:
- Gaji prioritas (ada deadline karyawan) [skor 2]
- Jadwal terstruktur [skor 1]
- Tahu konsekuensi telat gaji (UUCK Pasal 88) [skor 2]

### A6. Jika Kamu Salah Hitung Gaji
Tanya: "Kamu sudah bayar gaji, lalu sadar ada karyawan yang kelebihan bayar Rp 500.000. Apa langkahmu?"
Amati:
- Segera lapor atasan / HRD [skor 2]
- Hitung ulang dan konfirmasi [skor 1]
- Usul potong gaji bulan depan (dengan persetujuan karyawan) [skor 2]

## Bagian B — Soal Teknis Akuntansi & Perpajakan Gaji (Bobot 30%)

### B1. Jurnal Pencatatan Gaji (Soal Wajib)
Buat jurnal untuk transaksi berikut:
PT Maju Jaya membayar gaji karyawan bulan Juni 2024:
- Gaji bruto: Rp 75.000.000
- Potongan PPh 21: Rp 4.500.000
- Potongan BPJS Kesehatan (karyawan): Rp 750.000
- Gaji dibayarkan via transfer bank

Jawaban:

| Akun | Debit | Kredit |
|------|-------|--------|
| Beban Gaji | Rp 75.000.000 | |
| Hutang PPh 21 | | Rp 4.500.000 |
| Hutang BPJS Kesehatan | | Rp 750.000 |
| Kas / Bank | | Rp 69.750.000 |

Skoring:
- Debit Beban Gaji benar = 2
- PPh 21 di sisi kredit benar = 2
- BPJS Kesehatan (pot. karyawan) benar = 1
- Kas/Bank benar = 1

### B2. Jurnal Beban BPJS Perusahaan (Soal Wajib)
Perusahaan wajib membayar iuran BPJS Kesehatan 4% dan BPJS Ketenagakerjaan (JKK 0,24%, JKM 0,3%, JP 1%) dari gaji bruto. Jika gaji bruto Rp 75.000.000, buat jurnal beban BPJS perusahaan!

Jawaban:

Perhitungan:
- BPJS Kesehatan 4% = Rp 3.000.000
- JKK 0,24% = Rp 180.000
- JKM 0,3% = Rp 225.000
- JP 1% = Rp 750.000
- Total beban BPJS perusahaan = Rp 4.155.000

| Akun | Debit | Kredit |
|------|-------|--------|
| Beban BPJS Kesehatan | Rp 3.000.000 | |
| Beban JKK | Rp 180.000 | |
| Beban JKM | Rp 225.000 | |
| Beban JP | Rp 750.000 | |
| Hutang BPJS Kesehatan | | Rp 3.000.000 |
| Hutang BPJS TK | | Rp 1.155.000 |

Skoring:
- Perhitungan masing-masing benar = 3
- Jurnal debet/kredit benar = 2

### B3. Perhitungan PPh 21 Sederhana
Karyawan A (TK/0, belum menikah) memiliki gaji Rp 8.000.000/bulan. Hitung PPh 21 setahun!

Jawaban:
- Gaji setahun: 12 x Rp 8.000.000 = Rp 96.000.000
- PTKP TK/0 = Rp 54.000.000
- PKP = Rp 96.000.000 - Rp 54.000.000 = Rp 42.000.000
- PPh 21 terutang: 5% x Rp 42.000.000 = Rp 2.100.000/tahun
- PPh 21 per bulan: Rp 2.100.000 / 12 = Rp 175.000

Skoring:
- Gaji setahun benar = 1
- PTKP benar = 2
- PKP benar = 2
- Tarif 5% benar = 1
- Hasil akhir benar = 2

### B4. Perhitungan THR Prorata
Karyawan B mulai kerja 1 Maret 2024. Gaji Rp 6.000.000/bulan. THR Idul Fitri dibayarkan April 2024. Berapa THR yang diterima?

Jawaban:
- Masa kerja: Maret s.d. April = 2 bulan
- THR = (2/12) x Rp 6.000.000 = Rp 1.000.000

Skoring:
- Rumus prorata benar = 2
- Perhitungan benar = 2
- Sebut dasar aturan PP 36/2021 (pasal THR) = 1

### B5. Rekonsiliasi Utang Gaji
Saldo utang gaji awal Rp 5.000.000. Selama bulan Juni:
- Biaya gaji periode: Rp 75.000.000
- Pembayaran gaji: Rp 73.000.000
Berapa saldo utang gaji akhir?

Jawaban:
Saldo akhir = Saldo awal + Biaya gaji - Pembayaran
= Rp 5.000.000 + Rp 75.000.000 - Rp 73.000.000 = Rp 7.000.000

Skoring:
- Rumus benar = 2
- Hasil benar = 2

### B6. Accrual Gaji Akhir Periode
Jika gaji dibayarkan setiap tanggal 25, tapi tutup buku tanggal 30. Untuk gaji periode 25-30, bagaimana pencatatannya?

Jawaban:
Buat jurnal accrual (penyesuaian):

| Akun | Debit | Kredit |
|------|-------|--------|
| Beban Gaji (6 hari) | Rp XXX | |
| Utang Gaji | | Rp XXX |

Skoring:
- Paham konsep accrual = 2
- Jurnal penyesuaian benar = 2
- Menyebut 6 hari kerja = 1

## Bagian C — Excel Test Praktik (Bobot 35%)

### Soal 1: Data Entry Daftar Gaji (10 poin)
Buat tabel rapi dengan kolom: No, Nama, Gaji Pokok, Tunjangan, BPJS Karyawan, PPh 21, Take Home Pay
Isi data untuk 10-15 karyawan.
Amati: struktur tabel, format rupiah, kerapian.

### Soal 2: Rumus Take Home Pay (15 poin)
Rumus THP = Gaji Pokok + Tunjangan - BPJS Karyawan - PPh 21
Amati: formula benar + bisa di-drag ke bawah.

### Soal 3: IF Logic — Status Pajak (10 poin)
Jika PKP > Rp 0 maka "Kena Pajak", selain itu "Tidak Kena Pajak".
Amati: logika IF benar.

### Soal 4: VLOOKUP — Tarif PPh dari Tabel PTKP (15 poin)
Buat tabel di sheet terpisah: status PTKP, batas PKP, tarif.
Gunakan VLOOKUP untuk mengambil tarif pajak.
Amati: VLOOKUP dengan range lookup TRUE.

### Soal 5: Perhitungan THR Otomatis (15 poin)
Buat rumus THR prorata berdasarkan bulan masuk kerja.
Amati: rumus = (masa_kerja/12) * gaji_bulanan.

### Soal 6: Conditional Formatting (10 poin)
THP < Rp 3.000.000 = merah
THP Rp 3-7 juta = kuning
THP > Rp 7.000.000 = hijau

### Soal 7: Data Validation Dropdown (10 poin)
Dropdown untuk Status PTKP: TK/0, TK/1, TK/2, TK/3, K/0, K/1, K/2, K/3
Dropdown untuk Status Karyawan: Tetap, Kontrak, Magang

### Soal 8: COUNTIF / SUMIF per Departemen (15 poin)
Buat ringkasan: total gaji per departemen, jumlah karyawan per departemen.
Amati: COUNTIF dan SUMIF benar.
